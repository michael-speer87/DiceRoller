Overview -
Dice Roller app that was built using [a](https://www.youtube.com/playlist?list=PLsF3G8o7POZwo9bBmt8l9aFY9b94mJc6A) as a reference then modified to use jetpack compose,
have different dice and allow for numerous dice to rolled simultaniously.

Features -
Multiple Dice
  - D100
  - D20
  - d12
  - d10
  - d8
  - d6
  - d4
Multiple Dice Rolled at the same time
Images of all different dice
